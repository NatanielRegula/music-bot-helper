# zoxMusicBotHelper for [BD](https://github.com/BetterDiscord/BetterDiscord)

## What does it do?

It allows you to control a music bot that's in your vc.

## Prerequisites

- [bun](https://bun.sh/)

## Build

```bash
bun install
bun run build
```

## Live-update while developing

```bash
bun run build --watch
```
